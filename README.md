# Student Voice Platform (SSI UPH Analytics)

Transforming raw student satisfaction survey data into actionable intelligence for Universitas Pelita Harapan (UPH).

**Tech Stack:** Next.js 16 • React 19 • TypeScript • Supabase • Google Gemini AI • TailwindCSS 4

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
